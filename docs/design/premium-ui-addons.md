# Premium UI Add-On Blocks

Use these short blocks with the master prompt to quickly tune output behavior.

## Subtle Glass Add-On

Use subtle glass only as a secondary material:
- opacity low
- blur low
- thin border
- no glow
- no neon
- no rainbow
- never on primary reading surfaces

## Agnostic Premium Add-On

Avoid any domain-coded design cliches.
Do not visually signal finance, betting, SaaS, crypto, or agency aesthetics unless explicitly requested.
Keep the interface category-neutral and brand-led.

## Anti-AI-Smell Add-On

This must not look like a generic AI-generated premium page.
Win through restraint, spacing, hierarchy, and role-based typography.
Remove any decorative move that does not improve clarity.
