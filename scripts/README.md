# Scripts Directory Guide

- `scripts/backfill/`: Production backfill scripts for historical or missing data repair.
- `scripts/seed/`: Seed scripts that populate baseline reference datasets.
- `scripts/audit/`: QA, verification, and health-check scripts for pipelines/data.
- `scripts/analysis/`: Research and exploratory analysis scripts.
- `scripts/lib/`: Shared script utilities used by multiple script entry points.
- `scripts/archive/`: One-off debug/fix/regrade/repro scripts kept for traceability.
